# reactnative-mapbox-points-creation
Adds numeric points dinamically to a mapbox map by tapping.

## Getting started

`$ npm install`

Insert your MAPBOX_TOKEN in the 'src/App.tsx' file

MapboxGL.setAccessToken('MAPBOX_TOKEN');

## running on iOS

`$ cd ios && pod install`

`$ npm run ios`

## running on Android

`$ npm run android`
